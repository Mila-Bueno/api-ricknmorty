# api-ricknmorty
Projeto de consumo de API rest no tema do desenho Rick 'n Morty.

Para  utilizar basta colocar um número de id de personagem no input com o portal e abaixo escolher os filtros com as informações desejadas. Se não houver id do personagem ao clicar no botão de busca, ele deixará um aviso.

Projeto feito  através de tutorial do youtube: https://www.youtube.com/watch?v=NJzNc8-KqQI
Personalização de CSS feita por mim.
